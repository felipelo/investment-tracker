# Investment Tracker

A personal, local-first Canadian investment and Smith Maneuver tracker. It manages multiple portfolios, the accounts that move money through them (chequing, investment, margin, HELOC), holdings, dividends, and Adjusted Cost Base (ACB) for year-end tax filing, and presents a per-portfolio analytics dashboard.

This repository currently contains requirements only. See [REQUIREMENTS.md](REQUIREMENTS.md) for the full specification.
